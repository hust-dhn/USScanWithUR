# 功能函数

## 介绍

此项目主要是功能函数，可直接复用，注意使用时关注参数的个数与选取。

## 目录结构

项目目录结构如下：


- **function/**：包含功能函数
    **clear_files.py/**：该函数用于删除.jpg图片
    **copy_suffix.py/**：该函数用于将源文件夹中所有以"_1.jpg"结尾的文件复制到目标文件夹，并打印复制的文件数量
    **get_the_poses_from_cols.py/**：该函数用于从一个输入文件中读取内容，然后按照一定的规则（每隔三行）将特定行写入到输出文件中
    **inspect_ply_attrs.py/**：该函数用于分析和比较PLY点云文件中的顶点数据，特别是坐标和属性之间的关系
    **show_pcd.py/**：将点云的.pcd文件通过 python 可视化为点云图像
    **show_ply.py/**：将点云的.ply文件通过 python 可视化为点云图像
